# Emotion Recognition from Speech

## Objective
Detect human emotions from speech audio using machine learning and deep learning techniques.

## Features
- Upload audio files
- Extract audio features
- Predict emotions from speech
- User-friendly Streamlit interface

## Technologies Used
- Python
- TensorFlow
- Scikit-Learn
- Librosa
- Pandas
- NumPy
- Streamlit

## Installation

pip install -r requirements.txt

## Run the Application

streamlit run app/streamlit_app.py

## Required Libraries

- numpy
- pandas
- matplotlib
- seaborn
- librosa

- scikit-learn
